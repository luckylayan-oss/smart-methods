# ⭐ Star Keychain Tag

A simple 3D-printable star keychain tag, modeled in Onshape as my first CAD project.

## Description

A five-pointed star with a hole for a keyring, designed by tracing an
imported reference image in Onshape and extruding it into a solid part.

## Specifications

| Property        | Value            |
|-----------------|------------------|
| Overall size    | ~50 mm wide      |
| Thickness       | 2 mm             |
| Keyring hole    | 4 mm diameter    |
| Format          | STL (binary, mm) |

## Files

- `Part Studio 1.stl` — the 3D model, ready to slice and print

## Printing Suggestions

- **Material:** PLA (any color)
- **Layer height:** 0.2 mm
- **Infill:** 15–20%
- **Supports:** Not needed (flat part, prints directly on the bed)
- **Print time:** roughly 10–20 minutes

## How It Was Made

1. Imported a star image into Onshape and traced the outline with the Line tool
2. Added a 4 mm circle for the keyring hole
3. Extruded the sketch 2 mm to create the solid part
4. Exported as STL

## Tools Used

- [Onshape](https://www.onshape.com/) (free browser-based CAD)
