# Portfolio Website

A modern, responsive personal portfolio website built with React, Vite, and Tailwind CSS. Features smooth animations, dark/light mode, and a clean minimal design.

## Features

- Fast dev and optimized builds powered by Vite
- Sleek dark navy and mint teal palette inspired by top developer portfolios
- Dark / light mode toggle with preference saved to localStorage
- Fully responsive across mobile, tablet, and desktop
- Smooth page transitions and micro-interactions with Framer Motion
- Multi-page routing via React Router
- Working contact form using mailto submission
- Accessible, semantic HTML and keyboard-friendly navigation

## Tech Stack

- Framework: React 18
- Build Tool: Vite
- Styling: Tailwind CSS
- Routing: React Router
- Animations: Framer Motion
- Icons: Lucide React
- Fonts: Inter and Fira Code (Google Fonts)
